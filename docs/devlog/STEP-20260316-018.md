2. Mitigation: interval and policy thresholds are configurable (`--interval-seconds`, `--max-archives`, `--max-age-days`).
3. False-positive alerts can occur if retention thresholds are too strict for real traffic cadence.
4. Mitigation: retention policy is explicit and visible in generated report files for tuning.
